Wymagane do uruchomienia: 
•	.NET 8 SDK
•	Node.js
•	SQL Server

Kroki:
1.	Pobierz plik ZIP
2.	Rozpakuj go w dowolnym folderze
3.	Przejdź do folderu z rozpakowaną aplikacją, a następnie do /ReservationSystem
4.	W pliku appsettings.json należy dostosować swój connectionstring do bazy
5.	Uruchom za pomocą komendy dotnet run
6.	W konsoli wyświetli się adres pod jakim działa aplikacja
